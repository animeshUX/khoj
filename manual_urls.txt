# Manually-submitted listing URLs (e.g., from aunt/uncle texting you a link).
# - One URL per line.
# - Lines starting with # are ignored.
# - Optional: append " | note" to the URL for a submitter note (parsed but not displayed yet).
# - These bypass the price/distance/recency filters but still get scored.
# - Craigslist URLs get full parse+scoring; other sites fall back to JSON-LD + OpenGraph
#   metadata (price, lat/lng, address when the page exposes them).
#
# Example:
# https://newyork.craigslist.org/brk/apa/d/example/1234567890.html | Aunt Priya — nice kitchen
